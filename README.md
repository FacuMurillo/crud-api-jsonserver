# CRUD de Alumnos con JSON Server

## Descripción

Aplicación web desarrollada con HTML, CSS y JavaScript para gestionar alumnos mediante operaciones CRUD utilizando una API REST simulada con JSON Server.

---

## Tecnologías

* HTML
* CSS
* JavaScript
* JSON Server
* Node.js

---

## Funcionalidades

* Crear alumnos
* Mostrar alumnos
* Editar alumnos
* Eliminar alumnos
* Estado activo/inactivo

---

## Instalación

```bash
npm install
npm run server
```

Abrir `index.html` con Live Server.

---

## Endpoints

```http
GET /alumnos
POST /alumnos
PUT /alumnos/:id
DELETE /alumnos/:id
```
